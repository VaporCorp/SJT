== Description ==
 
A Free WordPress Blog / Magazine / Personal / Writers / Webshop. ClubNature is a clean minimal and responsive WooCommerce WordPress theme well suited for food, fashion, writers, travel, health, sports, fitness, business, finance, portfolio, design, art, photography, personal or any other creative websites, blogs and webshops. Developed with bootstrap framework that makes it mobile and tablet friendly. ClubNature has the option to add your social media links. ClubNature is ready for WooCommerce. Get free support at https://ask.thewpclub.com/

Theme Link: https://www.thewpclub.com/clubnature-wordpress-theme/
Demo Link : https://demo.thewpclub.com/clubnature/

== Installation ==
 
1. Upload the zip file to your themes directory.
2. Apply the theme.
3. Go to Appearance > Customize in main menu and configure your theme's settings.
